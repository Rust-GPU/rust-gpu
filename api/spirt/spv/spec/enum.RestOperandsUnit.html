Enum RestOperandsUnit

Source 
pub enum RestOperandsUnit {
    One(OperandKind),
    Two([OperandKind; 2]),
}
Expand description

Whether the trailing * “operand” (i.e. repeated arbitrarily many times), consists of just one operand, or two per repeat (used by e.g. OpPhi).
