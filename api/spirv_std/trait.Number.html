Trait Number

Source 
pub unsafe trait Number: Scalar { }
Expand description

Abstract trait representing a SPIR-V integer or floating-point type. Unlike Scalar, excludes the boolean type.

§Safety

Implementing this trait on non-primitive-integer or non-primitive-float types breaks assumptions of other unsafe code, and should not be done.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
